INSTALLATION ON YOUR SERVER VIA FTP

How To Install Via FTP :

Step 1 - Log into your hosting space via an FTP software (Filezilla for example).

Step 2 - Unzip your Download Pack file and use the extracted theme folder.

Step 3 - Upload the extracted theme folder into one folder or at the root of your server.

new Date(Date.UTC(2017, 3, 1, 0, 0));

Contact Form

You need to configure contact.php to get e-mails from the users.
Change those variables on line 11 with the real name & e-mail address to send the mail information: 

$from = 'Demo contact form - demo@domain.com';
$sendTo = 'Demo contact form - demo@domain.com';
$subject = 'New message from contact form';
